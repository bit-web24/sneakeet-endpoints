API Documentation

User Routes

Authentication

  • POST /signup
  • POST /login
  • POST /logout

Products

  • GET /products/:product_id
  • GET /products

Customer Profile

  • GET /account/:_id
  • PUT /account/:_id
  • DELETE /account/:_id

Cart

  • GET /account/:_id/cart
  • POST /account/:_id/cart
  • DELETE /account/:_id/cart/:item_id

Orders

  • GET /account/:_id/orders
  • POST /account/:_id/orders
  • DELETE /account/:_id/orders/:order_id

Favorites

  • GET /account/:_id/favorites
  • GET /account/:_id/favorites/:favorite_id
  • POST /account/:_id/favorites
  • DELETE /account/:_id/favorites/:favorite_id

Admin Routes

Products

  • GET /admin/products - Display all products
  • GET /admin/products/new - Display the form to create a new product
  • POST /admin/products - Create a new product
  • GET /admin/products/:_id - Display a specific product
  • GET /admin/products/:_id/edit - Display the form to edit a product
  • PUT /admin/products/:_id - Update a specific product
  • GET /admin/products/:_id/delete - Display the form to delete a product
  • DELETE /admin/products/:_id - Delete a specific product
